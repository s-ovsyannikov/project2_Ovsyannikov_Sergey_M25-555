# PrimitiveDB

Простая консольная СУБД для работы с таблицами.

## Управление таблицами

Доступные команды:

- `create_table <имя> <столбец1:тип> <столбец2:тип> ...` — создать таблицу.  
  Автоматически добавляется столбец `ID:int`.
- `drop_table <имя>` — удалить таблицу.
- `list_tables` — показать все таблицы.
- `help` — справка.
- `exit` — выход.

**Поддерживаемые типы:** `int`, `str`, `bool`.
